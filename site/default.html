About the default firmware

Includes all core Satellite1 features without radar-based presence detection. Use this if your device doesn't have an mmWave sensor installed.

Features

  • Voice assistant integration with Home Assistant
  • Wake word detection
  • Sendspin multi-room media player
  • Wi-Fi Improv for easy setup
  • OTA updates from stable and beta channels

Installation steps

  1. Connect your Satellite1 to your computer via USB
  2. Click the "Install Satellite1" button above
  3. Select the correct serial port when prompted
  4. Wait for the installation to complete
  5. Connect to the "satellite1" Wi-Fi network
  6. Configure your home Wi-Fi credentials
  7. Add the device to Home Assistant
Note: first-time installation erases the device. Existing ESPHome devices can be updated via OTA from Home Assistant.
