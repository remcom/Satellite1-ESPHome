About the HLK-LD2450

The HLK-LD2450 is an advanced 24GHz mmWave radar sensor with multi-target tracking. It can track up to 3 targets simultaneously and reports precise X/Y coordinates for each.

Detection Range
Up to 6 meters
Frequency
24GHz FMCW
Multi-Target
Up to 3 targets
Output
X/Y coordinates

Advanced features

  • Zone-based detection — define custom detection zones
  • Multi-target tracking with individual coordinates
  • Speed and angle detection per target
  • Configurable sensitivity and detection areas

Installation steps

  1. Connect your Satellite1 to your computer via USB
  2. Click the "Install Satellite1 LD2450" button above
  3. Select the correct serial port when prompted
  4. Wait for the installation to complete
  5. Connect to the "satellite1" Wi-Fi network
  6. Configure your home Wi-Fi credentials
  7. Add the device to Home Assistant
Note: first-time installation erases the device. Existing ESPHome devices can be updated via OTA from Home Assistant.
