What is Sendspin?

Sendspin is ESPHome's native protocol for synchronized multi-room audio, built into ESPHome core since 2026.6.0. Multiple Satellite1 units play audio in perfect sync throughout your home, with no external sync server to run or maintain.

Why an independent fork?

This fork tracks upstream ESPHome releases closely (currently 2026.7.0), with fixes for DAC power management, LED/timer logic, and the FUSB302B USB-PD driver landing as they're found. It's not a fork in maintenance limbo: releases are versioned, linted, and built from CI on every change.

Both this fork and the official FutureProofHomes firmware target the same hardware and now both build on Sendspin — pick whichever release cadence and maintenance style fits your setup.

A long-term goal here is upstreaming the Satellite1 hardware drivers into ESPHome core so any ESPHome user can use them directly. The PCM5122 DAC driver is already merged upstream; the TAS2780 amplifier driver is in progress.

Requirements

  • Google Chrome or Microsoft Edge (Web Serial required)
  • USB cable connected to your Satellite1
  • Home Assistant with Voice Assistant configured
  • ESPHome 2026.7.0 or newer for building from source

First-time installation

The device is erased during first install. After flashing, connect to the "satellite1" Wi-Fi network to configure your home network — no cables or CLI needed after that.
