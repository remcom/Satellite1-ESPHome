About HLK-LD2410

The HLK-LD2410 is a 24GHz mmWave radar sensor for human presence detection. It's ideal for detecting whether someone is in a room, even when stationary.

Detection Range
Up to 5 meters
Frequency
24GHz FMCW
Detection Type
Moving & Stationary
Output
Presence + Distance

Installation Steps

  1. Connect your Satellite1 to your computer via USB
  2. Click the "Install Satellite1 LD2410" button above
  3. Select the correct serial port when prompted
  4. Wait for the installation to complete
  5. Connect to the "satellite1" WiFi network
  6. Configure your home WiFi credentials
  7. Add the device to Home Assistant

Note: First-time installation will erase the device. Existing ESPHome devices can be updated via OTA from Home Assistant.
